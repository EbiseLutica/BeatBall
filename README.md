Beat Ball
==============
Beat Ball は，スポーツを題材にした音楽ゲームです．

使い方
------------
1. `$ git clone https://github.com/citringo/BeatBall`
2. Unityで開く

(私の)開発環境
------------
- Macbook Air 11 inch, Mid 2012
- macOS 10.13.2 High Sierra
- Unity 2017.2.1f1
- Visual Studio 2017 for Mac (Tools for Unity)
- git 2.15.1
